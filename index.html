Kết nối Google Sheets
Dán URL Publish CSV từ Google Sheets vào đây — dashboard tự đọc số liệu thực tế của bạn.
📋 Hướng dẫn lấy URL — làm 1 lần duy nhất
1
Mở file Marketing_Dashboard_Data.xlsx bằng Google Sheets (upload lên Drive → mở bằng Sheets)
2
Trong Google Sheets, nhấn menu File → Share → Publish to web
3
Chọn từng sheet theo thứ tự:
• Dropdown trái chọn tên sheet (vd: 📘 FANPAGE)
• Dropdown phải chọn Comma-separated values (.csv)
• Nhấn Publish → Copy URL → Dán vào ô tương ứng bên dưới
• Lặp lại cho từng sheet
4
Dán tất cả URL vào bên dưới → Nhấn Kết nối
⚡ Mẹo nhanh: URL Publish CSV có dạng:
https://docs.google.com/spreadsheets/d/ID/pub?gid=GID&single=true&output=csv
